# Android-APP-TutorMe

This App if an online platform for Tutors and Tutees to find each other.

* Designed an android platform called TutorMe to match tutors and tutees.
* Allows users to sign up and login as either tutor or tutee. The app matches tutor and tutee based on their subjects, time and locations. â€¢ Equipped the App with chatting and mapping function to help tutor and tutee get in touch.
